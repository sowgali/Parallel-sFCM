To get a matrix representation of an image use the "Image.ipynb" provided:
	1) Name your image as "test.jpg"
	2) It will output the matrix in "sample.txt"

To execute the sFCM files:
	1) For strategy 1 of simple load distribution use:
		"g++ <file>.cpp -o <out> -lpthread"

	2) For strategy 2 of helping algorithm:
		"g++ <file>.cpp -o <out> -lpthread -latomic"
